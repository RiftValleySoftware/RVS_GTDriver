RVS_GTDriver

This project is an open-source, low-level native Cocoa (iOS, macOS, watchOS and tvOS) BLE (Bluetooth Low Energy) driver for goTenna mesh network devices.

REQUIREMENTS

The Driver is provided as a Swift-only shared dynamic framework.

This is meant for iOS, macOS, watchOS, tvOS and ipadOS only.
