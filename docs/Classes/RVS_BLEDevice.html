RVS_BLEDevice

public class RVS_BLEDevice : NSObject

This class models a single discovered BLE device (in peripheral mode).

The class will offer a complete object model of the device to the API user. We need to abstract all Core Bluetooth stuff.

This class deliberately doesn’t conform to a sequence protocol, because we want to keep the details opaque.

Since this receives delegate callbacks from Core Bluetooth, it must derive from NSObject.
