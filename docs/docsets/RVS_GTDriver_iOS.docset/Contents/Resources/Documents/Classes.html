Main Driver Class
  • RVS_GTDriver

    This class implements the main skeleton of the driver API.

    The driver will always be a BT Central. It will scan for goTenna devices as peripherals, and instantiate internal instances of RVS_GTDevice for each discovered peripheral device.

    Since this receives delegate callbacks from CB, it must derive from NSObject.
